# Glossary

## Business Terms

- **Agent**: An AI configuration (voice, prompt, tools) that handles calls.
- **Channel**: A communication medium (Voice, SMS, WhatsApp).
- **Company / Organization**: A tenant in the system. Users belong to a company.
- **Member**: A user account belonging to a company.
- **Transcript**: The text representation of a voice conversation.
- **Usage**: Consumed resources (minutes, tokens) that are billed.

## Technical Terms

- **ASR (Automatic Speech Recognition)**: Converting spoken audio to text (STT).
- **JWT (JSON Web Token)**: Token used for stateless authentication.
- **LLM (Large Language Model)**: The AI model (e.g., GPT-4) generating responses.
- **MCP (Model Context Protocol)**: Standard for connecting AI models to external data and tools.
- **RAG (Retrieval-Augmented Generation)**: Enhancing LLM responses with external knowledge.
- **RLS (Row Level Security)**: PostgreSQL feature to restrict data access at the database row level.
- **SIP (Session Initiation Protocol)**: Protocol for voice over IP (VoIP) calls.
- **SSR (Server-Side Rendering)**: Rendering pages on the server before sending to the client.
- **TTS (Text-to-Speech)**: Converting text responses to audio.
- **Webhook**: A mechanism for one system to notify another of events (e.g., Vapi notifying CoreComm of a call).

## Acronyms

- **2FA**: Two-Factor Authentication
- **API**: Application Programming Interface
- **CRM**: Customer Relationship Management
- **KB**: Knowledge Base
- **MFA**: Multi-Factor Authentication
- **PR**: Pull Request
- **UI**: User Interface
- **UX**: User Experience
